Kodu çalıştırma talimatları
-------------------------------------------------- -------------------------------------------------- ---------------------------------------
1. En büyük gereksinim Python 3.6.1'dir. Bu, numpy ve tensor akış modülleriyle birlikte sisteme kurulmalıdır.
2. Dosyayı indirdikten sonra dosyayı herhangi bir klasöre yerleştirin.
3. Windows'ta (veya) Mac'te Terminal'de Komut İstemi'ni açın.
4. Dosyanın yerleştirildiği dizine gidin.
   c:\> cd dizin adı
5. Şimdi "python3 train_and_eval.py" komutunu kullanarak dosyayı çalıştırın.
6. Programın çeşitli adımlarda çıktısı birbiri ardına elde edilir. Yeniden Yapılandırma hatasındaki azalmayı ve Yeniden Yapılandırma hatasındaki artışı gözlemleyin. Program bitene kadar bekleyin.
